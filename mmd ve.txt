<mmdetection 가상환경 설정>

1. conda create -n mmdetection python=3.8

2. conda actiavate mmdetection

3. pip install torch==1.11.0+cu113 torchvision==0.12.0+cu113 torchaudio==0.11.0 --extra-index-url https://download.pytorch.org/whl/cu113

4. pip install mmcv-full==1.6.2 -f https://download.openmmlab.com/mmcv/dist/cu113/torch1.11/index.html

5. https://github.com/open-mmlab/mmdetection (다운로드 gitbash-clone 또는 zip 다운로드)

6. cd [mmdetection 경로] (다운로드 받은 mmdetection 폴더로 이동, activate 상태임)

7. pip install -r requirements/build.txt

8. pip install -v -e .

9.
